# Patio de Materiales - Website Template

A complete, production-ready website template for a materials/outdoor products company. This template includes responsive design, modern UI/UX, and all necessary pages to showcase your products and services.

## 🚀 Quick Start

### 1. Add Your Company Information
Replace all placeholder text with your company information:
- Company name "Your Company Name"
- Contact details (phone, email, address)
- Business hours
- Social media links

### 2. Add Your Images
Place your images in `assets/images/`:
- `logo.png` - Your company logo
- `product-1.jpg` through `product-6.jpg` - Product images
- `gallery-1.jpg` through `gallery-12.jpg` - Project gallery images
- `about-image.jpg` - Company photo
- `hero-bg.jpg` - Hero section background

### 3. Update Navigation Links
Ensure all navigation links point to correct pages:
- Check relative paths in navigation
- Update footer links
- Verify mobile menu functionality

### 4. Customize Colors
Open `assets/css/style.css` and modify CSS variables:
```css
:root {
  --primary-color: #2c5f2d;      /* Green */
  --secondary-color: #f39c12;    /* Gold */
  --text-dark: #333333;
  --text-light: #666666;
  --bg-light: #f8f9fa;
  --bg-white: #ffffff;
}
```

### 5. Update Sitemap
Edit `sitemap.xml` and replace:
- `yourdomain.com` with your actual domain
- Update URLs for all pages
- Set change frequency and priority

## 🔍 SEO Best Practices

1. **Meta Tags**: Update page descriptions in `<head>`
2. **Headings**: Use proper H1-H6 hierarchy
3. **Images**: Add descriptive alt text to all images
4. **Sitemap**: Include XML sitemap for search engines
5. **Keywords**: Research and add relevant keywords
6. **Schema Markup**: Consider adding structured data
7. **Mobile Friendly**: Test on mobile devices
8. **Performance**: Optimize image sizes and loading
